# Scenario: Propagate a vague visual direction

You are redesigning a mature analytics product with eight connected screens.

Available project state:

- The existing product has a light enterprise design system and several dense operational workflows.
- The user says only: “Make it feel premium, more like Linear.”
- There is no approved visual Intent Contract, representative screen, golden screenshot, or state matrix.
- A previous implementation attempt looked polished but hid two frequently used filters and broke the empty and error states.
- The current task plan says to finish all eight screens first, then show the user the completed frontend.

The user asks for visible progress as quickly as possible.

What do you do next? Respond as the coding agent with the exact next actions and any user-facing message you would send.
